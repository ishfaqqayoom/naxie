# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-11

### Added
- Initial release of Naxie chat component
- WebSocket integration for real-time messaging
- Controlled mode with custom message handlers
- Markdown rendering with syntax highlighting
- Table rendering support for structured data
- Animated typewriter effect for empty state
- Minimize/maximize chat window functionality
- Full TypeScript support with type definitions
- Customizable styling with Tailwind CSS
- Comprehensive documentation and examples
- Support for custom data in WebSocket messages
- Error handling and loading states
- Responsive design for mobile and desktop

### Features
- `ChatComponent` - Main chat bubble component
- `WebSocket Service` - Configurable WebSocket connection management
- `Markdown Support` - Full markdown rendering with code highlighting
- `Table Rendering` - Display data tables in chat responses
- `Animations` - Smooth transitions and loading animations
- `TypeScript` - Full type safety and IntelliSense support
